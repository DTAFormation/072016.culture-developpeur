Juillet 27, 2016 by Fryoyo

Presentation des Bases de données NoSQL

Les bases de données NoSQL sont des bases de données qui ne sont pas relationnelles. Exemple de bases de données NoSQL :

  • Mongo DB - oriente document
  • neo4J - oriente graphe
  • Cassandra,Hbase,Hypertable (inspirée de BigTable) - oriente colonne
  • Redis - oriente cle-valeur
